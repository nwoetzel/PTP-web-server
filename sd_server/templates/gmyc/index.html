This is a web interface for single threshold GMYC model implemented in Python. Please cite:
A General Species Delimitation Method with Applications to Phylogenetic Placements. Zhang, Jiajie, Kapli, P., Pavlidis, P., and Stamatakis, A. Bioinformatics (Oxford, England)(2013), 29 (22): 2869-2876
if you find it is useful to your research.

You can find the python implementation of the single threshold GMYC model in my GitHub repository. I encourage you to try the original R implementation, it has much richer features and has been widely used and tested. Be aware the results from Python and R implementation might differ slightly, this is due to different ways of parameter optimizations. Also note the input tree should be strictly ultrametric and bifurcating (with no zero branch lengths).
